# Precedente ontologia ISPRA

Questo modulo appartiene alla versione precedente dell'ontologia ISPRA ed è mantenuto qui per garantire la compatibilità con i dataset storici.

## [Ontologia Core](https://w3id.org/italia/env/onto/core)

Il modulo CORE definisce i concetti e le proprietà della precedente ontologia ISPRA (`ispra-core`). Al momento è ancora utilizzata per il [dataset RENDIS](https://w3id.org/italia/env/ld/rendis/dataset).

* **Namespace:** `https://w3id.org/italia/env/onto/core#`
